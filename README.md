# homebrew-opentraces

Homebrew tap for the [opentraces](https://opentraces.ai) CLI.

```bash
brew install JayFarei/opentraces/opentraces
```

This is the canonical tap for `opentraces`. The formula is updated
automatically on each release by the `update-homebrew.yml` workflow in
[JayFarei/opentraces](https://github.com/JayFarei/opentraces).

> Hitting `Error: Formulae found in multiple taps`? You have the legacy
> generic tap and this one both tapped. Run `brew untap jayfarei/tap`.
